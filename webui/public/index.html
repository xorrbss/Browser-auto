결재 미결함

그룹웨어에 로그인해 미결 결재를 스크랩·요약해 보여줍니다. 아래 목록에서 결재할 항목을 체크하고 [선택 항목 결재]를 누르면 한 번에 자동 결재됩니다(요약·내용을 보고 사람이 직접 선택). 브라우저 작업은 한 번에 하나씩.

⚡ 자동 승인 (고급 — 문서번호 직접입력)

⚠ 효과적: 실제 결재를 사람 확인 없이 자동 승인합니다(되돌릴 수 없음). 항상 먼저 미리보기(dry-run)로 확인하세요. 동기화된 문서만 가능(제목으로 내용 대조). 가드(전부 fail-closed): 유일 문서번호·제목 일치·금액 상한(선택)·결재선 승인전이 확인. 실제 승인엔 최대 건수 필수. Playwright 로그인 필요: node approve/auth-pw.mjs "<login>" "dashboard.office.hiworks.com" approve/hiworks.pw-state.json


				


				

						
🧾 승인 감사 로그

						
자동 승인 leaf가 매 단계 append-only로 기록한 감사 추적(data/approve-audit.jsonl) — 무엇을 요청/클릭/승인/건너뜀/재조정했는지의 출처. 최신순.

						
↻ 새로고침으로 감사 로그를 불러옵니다.

					


				

					
🔧 결재 캡처 — dry-run 테스트 (Gate-B)

					
새 시스템/양식(예: 합의)의 결재 동작을 폐기용 문서로 검증합니다. action(또는 직접 붙여넣은 블록)을 임시 미리보기 레시피로 dry-run하고 가드별 통과/실패를 보여줍니다 — 승인하지 않습니다(확인 전 정지). 레시피에 기록/활성화하지 않음(다음 단계). 비활성(enabled:false) action도 테스트 가능.
